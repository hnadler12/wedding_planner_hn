# 💍 Wedding Planner - API (Back-end MERN)
Une application de **Wedding Planner** construite avec le **stack MERN** (MongoDB, Express.js, React, Node.js).

🎯 **Objectif :** permettre aux utilisateurs de planifier leur mariage en ligne :
- Gérer les invités
- Suivre les prestataires
- Organiser leur budget
- Enregistrer les détails de l’événement

---

## 🏁 Étape 1 : Mise en place du Back-end

### 🔧 Technologies utilisées
- **Node.js**
- **Express.js**
- **MongoDB** (local ou Atlas)
- **Mongoose** (ODM)
- **dotenv** (variables d’environnement)
- **CORS**

---

## ⚙️ Initialisation du projet
- npm init -y
- npm install express mongoose cors dotenv
- Pour tester les routes et l'API :
Utilise Postman, Insomnia Thunder Client pour tester les routes :

GET http://localhost:5000/api/weddings

POST http://localhost:5000/api/weddings

PUT http://localhost:5000/api/weddings/:id

DELETE http://localhost:5000/api/weddings/:id
